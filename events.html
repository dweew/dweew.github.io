Content Type: Events

CSS · HTML · Bootstrap · Flickity · Dreamweaver · CMS

“Events” was a content type I designed for VCU's Terminal4 content management system. It’s purpose was to display google calendar events in a more user friendly manner. Specifically, this content type was designed for department sites and would be customizable to their upcoming events. Due to COVID-19, this project will remain incomplete.

There are three variations of “Events”:

  1. Slider View see demo
  2. List View see demo
  3. Detailed View see demo
see demo

Slider View

The image above shows the slider view. This variation is how the content type would be displayed on web pages. It’s responsive and relies on Flickity’s javascript to swipe. Selecting the more events link would link to the list view.

List View

The image above shows the list view. This variation shows a list of all upcoming events. The plan was to have a designated web-page for list view.
