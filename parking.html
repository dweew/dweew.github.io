Re-design: Freshman Parking

Illustrator · Photoshop · CMS

This project served to re-design parts of the old VCU Freshman Parking page with the three main goals in mind:

  1. Create a banner for the header
  2. Edit the wordy web content
  3. Re-arrange content types

Goal 1: Banner

The banner above was created in Adobe Illustrator with the use of the pattern tool and icons from VCU’s icon library. Adobe Photoshop was used to crop the banner at the following pixel widths: 576, 768, 992, 1200, 2560. By doing so, I had more control on how the pattern displayed at different breakpoints.

view full webpage

Goal 2: Web Content

The purpose of this web-page is to inform the reader, likely freshman students or their parents, of freshman parking policies. With the purpose and audience in mind, I took some liberties in simplifying the language and shortening sentences.

Goal 3: Content Types

The old freshman parking page had outdated information; it listed expired links and advertised services VCU no longer offered. It also used outdated content types, so the webpage was not consistent with VCU’s newer web pages. I removed outdated information, copied relevant text into new content types, and re-arranged content types with user friendly practices in mind.
